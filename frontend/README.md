# Frontend - AI Food Supply Chain Optimizer

This is the frontend application for the AI-Driven Food Supply Chain Optimizer, built with React and Vite.

## Features (Planned)
- Computer Vision Shelf-Life Predictor (using device camera)
- Dynamic Pricing Engine interface
- B2B Surplus Marketplace
- Vendor Analytics Dashboard
- Offline-first capabilities
- Multi-language support (English/Swahili)

## Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack
- React 18
- Vite
- (Planned) TensorFlow.js for on-device ML
- (Planned) Socket.io for real-time updates
- (Planned) Material-UI or Ant Design for components
